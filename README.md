## 메모마스터 ##

안드로이드로 만든 2번째 앱입니다.
사용된 기능은 다음과 같습니다.

* 처음 시작때 Splash
* ListView
* Email & 첨부
* JSON Parsing
* TabLayout
* Fragment
* SQL
* File input output
* MediaPlayer & Voice REC
* Permission

연습용으로 만들었지만 개인적으로 사용할 목적으로 만들었습니다.
메모를 저장한후 뒤로 가기를 하면 Sql에 저장이 됩니다.
텍스트 작성시 임시저장으로 Shared를 사용하여 실수로 지우는 것을 방지하였으며
빈칸의 경우 저장을 방지하게 됩니다.
